# KA-Vehicle-PUC-Check

KA-Vehicle-PUC-Check is a lightweight lookup app for Karnataka vehicle PUC records. It accepts a vehicle registration number, checks Petrol first, falls back to Diesel automatically, and shows the matched record set directly in the app.
